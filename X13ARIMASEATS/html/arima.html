MODELO ARIMA

En esta sección se confecciona la modelación, donde se pueden identificar tantos modelos plausibles como se desee; para ello dispone de ciertos elementos que fueron retomados del programa X-13 ARIMA-SEATS. La demarcación en rojo contiene el editor del SPC, aquí podrá modificar las especificaciones del modelo a testar (recuerde cuidar la sintaxis, véase el manual previamente referido); en azul se muestra el archivo output, el cual integra áridamente todas las salidas: valores, pruebas, estadísticos, gráficos; en rosa encontrará una gama de gráficas ilustrativas; en café es donde podrá descargar toda la información producida por el proceso de modelación, cada archivo se guarda con una extensión única, pero pueden ser abiertos como un usual archivo TXT; en naranja tiene el diagnóstico raudo con estadísticos e información del modelo ejecutado; en amarillo se añade un botón que devuelve el modelo inicial en el editor, correspondiente al SPC cargado en la pestaña "INICIO"; en verde se localiza el botón de ejecución del programa, que generará todos los elementos mencionados para el modelo modificado en el editor.

El conjunto del diagnóstico, las gráficas y el archivo output, le proveerá los fundamentos necesarios para dictaminar si el modelo probado es estadísticamente válido o no. Se puede ajustar más de un modelo a una serie de tiempo, mientras se cumplan los estándares estadísticos.
